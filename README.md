# スマートキャッシュ

先払い買取（商品券・ギフトカード買取）の比較サイト

## 概要

スマートキャッシュは、商品券やギフトカードの先払い買取業者を徹底比較するWebサイトです。

### 主な機能

- 業者ランキング（総合、最短振込、高換金率、安全性重視）
- 業者詳細ページ
- 基礎知識ガイド
- 安全ガイド（悪徳業者の見分け方、違法性について）
- 目的別・商品別・地域別ページ
- FAQ
- コラム・お役立ち情報

## 技術スタック

- Next.js 16.2
- React 19
- TypeScript 5
- Tailwind CSS 3.4

## 開発

```bash
# 開発サーバー起動
npm run dev

# ビルド
npm run build

# 本番環境起動
npm start
```
